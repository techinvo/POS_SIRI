การตั้งค่า Google Sheets

URL ของ Google Apps Script ที่สร้างขึ้น

วิธีการตั้งค่า:

  1. สร้าง Google Sheets ใหม่ด้วยคอลัมน์: id, name, barcode, price, stock
  2. ไปที่ Extensions → Apps Script
  3. คัดลอกโค้ด Google Apps Script ด้านล่างไปวาง
  4. Deploy เป็น Web app และคัดลอก URL มาใส่ด้านบน
  5. กดทดสอบการเชื่อมต่อ

ตั้งค่ากล้องมือถือ

ติดตั้ง "IP Webcam" บนมือถือ → Start Server → ใส่ URL ที่ได้ + /shot.jpg (ภาพเดี่ยว) หรือ /video (สตรีม)

วิธีใช้กล้องมือถือ:
  1. ติดตั้ง "IP Webcam" บนมือถือ
  2. เชื่อมต่อ WiFi เดียวกันกับ PC
  3. เปิดแอป → กด "Start Server"
  4. คัดลอก IP + เพิ่ม "/shot.jpg"
  5. ตัวอย่าง: http://192.168.1.100:8080/shot.jpg
  6. วางใน URL Box ด้านบน
